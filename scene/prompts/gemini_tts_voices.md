## Technical Integration Note: Live Streaming & Production (Larix)
While **Larix** (specifically *Larix Broadcaster*) is an external tool rather than an API configuration within the Gemini engine itself, these prebuilt voices are built for extreme low-latency processing. When setting up real-time voice synthesis for interactive platforms, automated broadcast alerts, or live streaming pipelines, you can route the generated audio streams seamlessly directly into Larix using low-overhead transport protocols like SRT or RTMP.
